# webhook-handler
各種Webhook対応サービスをつなげる簡易プロキシ

## 使用方法
```sh
git clone https://github.com/KisaragiEffective/webhook-handler
cd webhook-handler
cargo run
```

## 使用言語/フレームワーク
* [Rust](https://www.rust-lang.org)
* [actix-web](https://actix.rs)

## 対応サービス
* [Todoist](https://todoist.com)
* [Discord](https://discord.com)
